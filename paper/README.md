## Current Status

**Target Venue:** TMLR (rolling deadline, submit ~March 2026)
**Review Status:** Ready for Round 0 (3-reviewer system)
**Paper Length:** 18 pages (no page limit for TMLR)

### Key Results
- 3-9% accuracy gap between ternary and FP32 on CIFAR
- Augmentation widens gap: FP32 benefits 1.5-3.7× more
- conv1 accounts for 54-74% of gap (only 0.08% params)
- Recipe (FP32 conv1 + KD): 89% recovery on CIFAR-10, exceeds FP32 on harder tasks

## Review Workflow (Simplified)

### 3-Reviewer System

Each round uses **3 personas** to simulate diverse reviewer feedback:

1. **ML Theory Researcher** - Statistical rigor, theoretical grounding
2. **Quantization Expert** - Field knowledge, baseline comparisons
3. **Systems Practitioner** - Reproducibility, deployment practicality

### Per-Round Process

```bash
# Round N workflow
cd paper/tmlr && make

# Get 3 independent reviews:
# - Paste paper + persona_1 prompt to Claude → save to round_N/reviewer_1.md
# - Paste paper + persona_2 prompt to Claude → save to round_N/reviewer_2.md
# - Paste paper + persona_3 prompt to Claude → save to round_N/reviewer_3.md

# Read all 3 reviews
cat reviews/tmlr/round_N/*.md

# Create implementation plan (in PLAN.md or similar)
# - Synthesize MAJOR/MINOR issues across all 3 reviews
# - Prioritize: fix MAJOR issues first, MINOR if time permits
# - Document decisions

# Fix issues in main.tex

# Rebuild and repeat
make clean && make
```

## Tips

### Building the Paper
```bash
cd paper/tmlr
make           # Build PDF
make clean     # Remove build artifacts
```

### Common Issues

**"No rule to make target"**
- Check that preamble.tex, macros.tex, bibliography.tex exist in `tmlr/`
- Ensure figures/ and tables/ directories exist in `tmlr/`

**"File not found" in LaTeX**
- Verify `\input{}` statements use local paths (no `../`)
- Check `\graphicspath{{figures/}}` points to local directory

**Stale tables/figures**
- Re-run analysis scripts after new experiments
- Check that scripts write to `paper/tmlr/tables/` and `paper/tmlr/figures/`
